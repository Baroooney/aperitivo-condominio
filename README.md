# Aperitivo Condominio

Web app mobile-first per organizzare l'aperitivo di condominio del 2 luglio.

## Funzioni

- lista condivisibile di cose da portare
- assegnazione per nome o famiglia
- filtri per categoria e voci libere
- suggerimenti per bilanciare bere, mangiare ed extra
- condivisione dello stato tramite link

## Pubblicazione

Il progetto e' statico: GitHub Pages puo' pubblicarlo direttamente da `index.html`.
